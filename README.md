# 🍺 La Taverna dello Svincolato

> *Le squadre cambiano ogni stagione. Gli Allenatori restano.*

## Cos'è

La Taverna dello Svincolato è una piattaforma web che affianca il Fantacalcio con un sistema di progressione RPG.

L'obiettivo non è sostituire il Fantacalcio, ma estenderlo attraverso spedizioni, crescita dell'Allenatore, collezionismo e narrazione emergente.
