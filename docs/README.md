# Struttura

## Game Design Document (GDD)

Descrive il gioco dal punto di vista del giocatore.

Contiene:

- Visione
- Gameplay
- Progressione
- Compagnia
- Spedizioni
- Regioni
- Economia
- Stagioni

## Technical Design Document (TDD)

Descrive l'architettura software.

Contiene:

- Software Architecture
- Domain Model
- Behavior Model
- Engine Architecture
- Application Services
- Database Model
- External Integration
- Technical Infrastructure
